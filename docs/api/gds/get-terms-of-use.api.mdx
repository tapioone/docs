## Get Terms Of Use



Provides the Terms Of Use for the given Applications in the given locales.
If no applicationId is given Terms Of Use for all applications will be provided.
If there are no Terms Of Use for given applicationId the application will be ignored.
If no locale is given Terms Of Use for all available languages will be provided.
If there are no Terms Of Use for given locale the response will fallback to the english URL but keep the requested locale.
